Example 1: break

Use a div with a class of "break", which we define as flex-basis: 100%, (basis can be thought of as "default amount of space" + whatever else is done to it)

Here, we're making an empty div with the class of 'break' with a default of 100% space taken up with a height of 0. this creates a new "row"

We allow "wrapping" on the container
