Grid 2:

Specifying row heights and # of rows, width of cols & # of cols

In grid2 we set specified the heights of the first two rows the be 100px and 150px. We also set three columns with "fractional" units. We see two 1 part items, and a 2 part item. This means the 2 part column takes up twice as much space as the 1 part column.

We have two rows and 3 columns, meaning 3 columns per row, which is all six of our items in the grid. AFTER the grid has used up all it's defined areas, it will start adding them at the bottom. So new items will be added to the next row with the same amount of columns as above.
